## Task 4: Ablation Study A-E

**Type:** implementation  
**Files:** `notebooks/07_Evaluation_Ablation.ipynb`

Implement ablation variants with honest evidence type labels:

### Full Model

- Use Phase 5 `final_ensemble` test metrics.
- Evidence type: `direct_artifact`.

### Model A: Remove PSO

- Direct branch comparison:
  - `dl_baseline` vs `dl_pso`.
- If selected Phase 5 blend uses non-zero DL weight:
  - Recompute the selected blend with `dl_baseline` replacing `dl_pso`.
  - Keep XGBoost/LightGBM weights unchanged.
- If selected blend has zero DL weight:
  - Record that full-ensemble No-PSO delta is not meaningful and use branch-level delta.
- Evidence type:
  - `direct_artifact` or `direct_artifact_branch_level`.

### Model B: Remove PCA

- Controlled ablation:
  - Train LightGBM on raw fused features (`features_raw_*`) under the same seed/config as PCA LightGBM.
  - Compare against LightGBM on PCA features.
- Record memory/time where possible.
- If raw feature training is too slow:
  - Use stratified sample and label `resource_fallback`.
- Evidence type: `controlled_lightgbm`.

### Model C: Remove Advanced Behavioral Features

- Use Phase 2 metadata to identify:
  - 768 BERT embedding dimensions.
  - 5 basic behavioral features.
  - 4 advanced behavioral features.
- Create BERT + 5 basic behavioral feature matrix.
- Apply PCA/SVD fit on train only if dimensionality reduction is needed.
- Train the same controlled LightGBM config.
- Compare to controlled LightGBM with all 9 behavioral features.
- Evidence type: `controlled_lightgbm`.

### Model D: Remove Ensemble

- Use Phase 4 `dl_pso` test metrics.
- Evidence type: `direct_artifact`.

### Model E: Baseline

- Use Phase 4 `dl_baseline` test metrics.
- Evidence type: `direct_artifact`.

For all variants:

- Save metrics with:
  - `ablation_variant`
  - `removed_component`
  - `evidence_type`
  - `seed`
  - `split`
  - `threshold`
  - `threshold_strategy`
  - all core metrics.
- Compute deltas vs Full Model:
  - `delta_macro_f1`
  - `delta_precision_fake`
  - `delta_roc_auc`
  - `delta_pr_auc`
- Save:
  - `reports/tables/phase7_ablation_results.csv`
  - `reports/tables/phase7_ablation_delta.csv`
  - `reports/figures/phase7_ablation_delta.png`

**Acceptance criteria:**

- EVAL-03 and EVAL-04 are satisfied.
- Ablation table answers which component helps and by how much.
- Controlled/surrogate ablations are clearly labeled to avoid overclaiming.
